# AI Driven Intelligent Doctor Recommendation and Real Time Appointment System

## Description
This project is a smart healthcare platform that helps patients find the most suitable doctor based on symptoms, location, ratings, and availability.

## Technologies Used
- Machine Learning
- Cloud Computing
- Cyber Security
- Full Stack Development

## Features
- AI Doctor Recommendation
- Real Time Appointment Booking
- Emergency Priority System
- Automated Reminders
- Secure Patient Data Storage
